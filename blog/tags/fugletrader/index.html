如何使用 TradingView 設定訊號交易？

2024年9月29日 · 閱讀時間約 5 分鐘
Kevin Wang
Sr. Backend Engineer @ Fugle

TradingView 是一個強大的線上金融市場分析平台，提供多樣化的圖表工具、技術分析指標、交易信號及社群互動功能。透過其 快訊 (Alerts) 與 Webhooks 功能，使用者可以設立警示或執行交易策略。當這些警示或策略被觸發時，TradingView 會向 FugleTrader API 發送一則包含股票代號、交易方向、價格、數量等資訊的 Webhook 訊息，讓 FugleTrader 自動執行交易。
